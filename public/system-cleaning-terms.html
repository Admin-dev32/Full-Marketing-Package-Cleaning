Términos y Políticas

Al contratar un paquete de Bako Systems aceptas estos términos para proyectos de sistemas y marketing para negocios de limpieza.

Vigente
Uso de APIs e infraestructura

Cuándo podría ajustarse la capa técnica

El sistema usa APIs para calendarios, automatizaciones y reservas. El uso normal está incluido.

  • Si el tráfico o las automatizaciones crecen de forma inusual, puede requerirse infraestructura extra o una suscripción adicional.
  • Cualquier ajuste se comunicará por adelantado y aplicará solo a periodos futuros.
  • El objetivo es sostener la estabilidad del sistema cuando la demanda aumenta.
Cambios de precios

Ajustes por plataformas o alcance

Las tarifas pueden actualizarse por costos de plataformas, inflación o evolución del servicio.

  • Cualquier cambio se avisará con anticipación y no aplica a periodos ya facturados.
  • Buscamos mantener la relación costo-beneficio alineada con el valor entregado.
Alcance del servicio

Sistema operativo y marketing por fases

Cada paquete incluye un setup inicial y, según el nivel, componentes de marketing. Todo se documenta en una propuesta o SOW.

Entregables principales

  • Setup del sistema según paquete (web, reservas, pagos, Google Business, automatizaciones).
  • Campañas y contenidos según el nivel contratado y los add-ons seleccionados.
  • Acceso y capacitación básica para usar el sistema sin depender de código.
Pagos y suscripciones

Setup, mensualidades y add-ons

Los montos se muestran antes de pagar. El checkout seguro se procesa en Stripe.

  • El pago inicial incluye setup + primer mes de mensualidad y cualquier add-on seleccionado.
  • Las mensualidades siguientes incluyen la parte mensual de add-ons y el presupuesto de Meta Ads indicado.
  • El presupuesto de Google Ads se paga directo a Google con tu tarjeta; no se cobra en nuestro checkout.
  • Las suscripciones se renuevan mensualmente; puedes solicitar ajustes antes del siguiente ciclo.
Publicidad y contenidos

Meta Ads, Google Ads y producción

Las campañas requieren aprobaciones de texto/creativos y acceso a las cuentas publicitarias.

  • Necesitamos acceso a tu Business Manager/Ad Accounts y facturación activa para publicar anuncios.
  • El presupuesto de Meta Ads se factura a través de nuestra suscripción; Google Ads se paga directo a Google.
  • Los creativos producidos para tu campaña pueden usarse en tus canales mientras la cuenta esté activa.
Cambios y cancelaciones

Ajustes razonables y fin de servicio

Buscamos flexibilidad, pero ciertos límites aplican para mantener el ritmo del proyecto.

  • Incluimos iteraciones razonables dentro del alcance; cambios mayores pueden cotizarse como add-on.
  • Las cancelaciones de mensualidad aplican al siguiente ciclo; no hay reembolsos de periodos ya trabajados.
  • Si se pausa el proyecto, “Keep System On Only” mantiene el sistema en línea (sitio, reservas básicas e integraciones) sin campañas ni gestión activa.
Datos y accesos

Confidencialidad y resguardos

Usamos los accesos que nos compartas solo para operar tu sistema y campañas.

  • No compartimos tus accesos con terceros ajenos al proyecto.
  • Puedes revocar accesos cuando termine el servicio; recomendamos mantener respaldos.
  • Las métricas y reportes son confidenciales y se usan solo para optimizar tu operación y marketing.
Contacto

Cómo comunicar cambios o soporte

Usamos canales definidos para mantener trazabilidad de solicitudes y entregables.

  • El canal principal se acuerda al inicio (email o chat de proyecto) para solicitudes y actualizaciones.
  • Las solicitudes urgentes se atienden dentro de horarios hábiles acordados.
  • Para los primeros meses puedes activar Plan Protection: si no se cumplen mínimos acordados, pagas solo una cuota reducida; si se cumplen, aplica la mensualidad completa más la protección.
Glosario / Índice de términos

Significados clave en este sistema

Definiciones simples para entender cómo funciona la capa técnica y de marketing.

  • API: interfaz para que el sistema se conecte con otros servicios (calendarios, pagos, automatizaciones).
  • Endpoint: URL específica que recibe o envía datos entre el sistema y otros servicios.
  • Automatización: flujo que ejecuta acciones (recordatorios, confirmaciones) sin intervención manual.
  • Stripe: plataforma usada para procesar pagos de forma segura.
  • Checkout: pantalla de pago donde el cliente confirma sus datos y autoriza cargos.
  • Suscripción: cobro recurrente que se ejecuta mes a mes según el servicio contratado.
  • Meta Ads: publicidad en Facebook e Instagram para atraer clientes al sistema.
  • Google Ads: anuncios en Google para personas que buscan servicios de limpieza.
  • Hosting: servicio que mantiene el sitio y el sistema disponibles en línea.
  • Dominio: la dirección web (URL) donde los clientes acceden al sistema.
